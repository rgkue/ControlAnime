Formatos soportados
JSON Export de ControlAnime
El archivo que exportas desde la sección Exportar. Importación perfecta con categorías y episodios.
XML MyAnimeList
Ve a MAL → Perfil → Export → "Export Anime List". El sistema detecta automáticamente Completed, Watching, Plan to Watch.
XLSX Excel / CSV
Cualquier hoja con una columna "Titulo" o "Title". Opcionalmente "Status" y "Episodes Watched".
TXT Texto plano
Un título por línea. Se importarán todos como "Visto". Útil para listas simples copiadas de notas.
